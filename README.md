# Mini Account Book

## 📌 프로젝트 소개

Mini Account Book은 **개인 가계부 기록을 간단하고 빠르게 작성**할 수 있도록 만든 웹 애플리케이션입니다. 수입과 지출을 기록하고, 월별 내역을 한눈에 확인할 수 있도록 구성된 초간단 가계부입니다.

---

## 🧩 구현한 기능

* **수입/지출 등록**: 금액, 카테고리, 날짜를 입력해 기록
* **내역 리스트 표시**: 입력한 내역을 화면에서 즉시 확인 가능
* **월별 필터링 기능**: 특정 월의 가계부 내역만 보기
* **내역 삭제 기능**: 작성한 항목 개별 삭제 가능
* **데이터 로컬 저장(LocalStorage)**: 새로고침해도 데이터 유지

---

## ▶️ 실행 방법

1. 이 프로젝트를 로컬로 다운로드하거나 GitHub에서 Clone합니다.
2. 프로젝트 폴더를 연 뒤 **`index.html` 파일을 브라우저로 실행**합니다.
3. 별도의 서버 환경 없이 **브라우저만 있으면 바로 실행 가능**합니다.

---

## 🛠 사용한 기술 스택

* **HTML5** – 화면 구조 구성
* **CSS3** – 기본 UI 스타일링
* **JavaScript (Vanilla JS)** – 기능 구현 및 이벤트 핸들링
* **LocalStorage API** – 데이터 저장
